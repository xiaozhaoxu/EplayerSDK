# eplayersdk
eplayersdk


集成方式
Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
Step 2. Add the dependency

Subproject 
	dependencies {
	        compile 'com.github.xiaozhaoxu.eplayersdk:SDKLibrary:0.1.3'
	}
